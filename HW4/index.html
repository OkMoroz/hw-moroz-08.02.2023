Перша частина завдання

Селектор - це частина CSS-правила, яка повідомляє браузеру, до якого елементу веб-сторінки буде застосований CSS стиль. Тобто, селектор - це вибірка та формальний опис того елемента чи групи елементів, до яких будуть застосовані CSS стилі.

Існує 7 видів селекторів:

  1. Селектори тегів
  2. Селектори класу
  3. Селектор по ID
  4. Універсальний селектор
  5. За атрибутами
  6. Селектори псевдокласів
  7. Селектори псевдоелемент

Селектори:

  • class - обирає елемент, що має вказаний клас.

  • #id - здійснює вибірку по ідентифікатору елемента.

  • all - Селектор * обирає всі елементи. Також селектор * може обирати всі елементі всередені іншого елементу.

  • element - здійснює вибірку всіх елементів із зазначенним іменем..

  • adjacent - використовується для вибірки елементів element2, що розміщені відразу після елементів element1.

  • first-letter - задає стиль першої літери визначеного елемента.

  • hover - використовується для вибірки та задання стилю посиланням, на які наведено курсор.

  • in-ranger - обирає, ті елементи форми, значення яких знаходиться в межах дозволеного діапазону.

  • link - застосовується до посилань, які ще не відвідувалися користувачем та задає їм стильове оформлення.
